# 무한매수 다이어리 — 설치 및 사용 안내

## 먼저 알려드릴 점
Claude가 작업하는 환경은 인터넷(네트워크)이 차단되어 있고 Android SDK도 없어서, 여기서 서명된 `.apk` 파일을 직접 컴파일해 드릴 수는 없습니다.
대신 **스마트폰 브라우저에서 실제 앱처럼 설치해서 쓸 수 있는 웹앱(PWA)** 형태로 만들었습니다. 아이콘을 눌러 실행하면 주소창 없이 전체 화면으로 열리고, 오프라인에서도 동작하며, 데이터는 휴대폰에 저장됩니다. 아래 A안으로 충분히 "앱처럼" 쓰실 수 있고, 진짜 apk 설치 파일이 꼭 필요하시면 B안을 참고하세요.

---

## A안 — 바로 쓰기 (추천, 5분)
PWA는 파일을 그냥 더블클릭해서 여는 게 아니라, 아주 간단한 무료 호스팅에 올려서 주소로 접속해야 홈 화면 설치와 오프라인 기능이 정상 동작합니다.

1. **무료 호스팅에 올리기** (아래 중 아무거나, 계정 없이도 가능한 것도 있음)
   - [Netlify Drop](https://app.netlify.com/drop) : `무한매수_다이어리_앱` 폴더를 통째로 브라우저에 드래그&드롭만 하면 즉시 주소가 생성됩니다.
   - GitHub Pages, Vercel 등도 동일한 방식으로 사용 가능합니다.
2. 생성된 주소(예: `https://xxxx.netlify.app`)를 **안드로이드 폰의 Chrome**으로 접속합니다.
3. Chrome 오른쪽 위 점 3개 메뉴 → **"홈 화면에 추가"** (또는 자동으로 뜨는 "앱 설치" 배너) 를 누릅니다.
4. 홈 화면에 아이콘이 생기고, 실행하면 주소창 없이 전체 화면 앱처럼 열립니다. 이후 비행기 모드에서도 사용 가능합니다.

## B안 — 진짜 .apk 설치 파일이 꼭 필요할 때
A안으로 올린 주소가 있다면, [PWABuilder](https://www.pwabuilder.com) 에 그 주소를 입력 → "Android" 패키지 생성을 누르면 실제 설치용 `.apk`/`.aab` 파일을 무료로 만들어줍니다. (Anthropic/Claude와 무관한 별도 무료 서비스입니다.)

## 앱 사용법
- **대시보드**: 3개 증권사(토스/삼성/키움) 전체 합산 평가금액·손익을 확인
- **증권사 탭**: 종목 추가, 종목 카드 목록 확인
- **종목 상세**: 총 시드 · 현재가 수정, **"매수 입력" / "매도 입력"** 버튼으로 회차별 거래 기록
  - 매수 시 회차가 자동으로 1씩 누적되고(최대 40회), 매도 시 가중평균 원가법으로 평균단가·실현손익이 자동 계산됩니다
  - 매도 후 확보된 현금은 "남은 시드"에 자동으로 반영되어, 이어서 새로운 회차 매수에 사용할 수 있습니다
- **설정 탭**: 데이터 JSON 백업/복원, CSV 내보내기(엑셀 파일과 호환), 전체 초기화

## 데이터 관련 유의사항
- 모든 데이터는 **해당 브라우저(기기) 안에만** 저장됩니다(서버 전송 없음, 로그인 불필요).
- 브라우저 데이터를 지우거나 기기를 바꾸면 데이터가 사라지므로, 설정 탭에서 **주기적으로 "백업 파일 내보내기"** 를 눌러 JSON 파일을 보관해 두는 걸 권장합니다.
- 처음 실행 시 첨부하셨던 엑셀의 토스증권 TQQQ·SOXL 기존 보유분이 1회차 매수 기록으로 미리 채워져 있습니다. 필요 없으면 삭제 후 새로 입력하시면 됩니다.
